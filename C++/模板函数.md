模板函数是程序员为一个功能的函数写出的，通用的代码模式，然后可以将其用于许多不同的类型。

## 解释
```
int max(int x, int y) { return (x > y) ? x : y; }

double max(double x, double y) { return (x > y) ? x : y; }
```

这两个函数的功能是一样的，实现逻辑也是一样的，只是处理的数据类型不同，因此可以将其模板化

```
template <typename ValueType>
ValueType max(ValueType x, ValueType y)
    { return (x > y) ? x : y; }
```
这就是一个max函数的**模板函数**。

在这个定义中，标识符**ValueType**是调用max时使用的参数类型的**占位符**。typename关键字告诉C++编译器==这个占位符表示类型的名称==，从而使编译器能够正确地解释该标识符。


```
max('A', 'Z');

等价于下面这样

char max(char x, char y)
    { return (x > y) ? x : y; }
```

==每当编译器遇到将模板函数应用到它尚未见过的类型时，它就生成一个适用于该类型的函数体的全新副本==。所以把这个策略叫做“[[模板]]”。

## 实例化

==把模板函数落实到具体的数据类型==，就是实例化，有两种实例化方式。

```
template <typename T>      // 声明：T 是待填的类型
T add(T a, T b) {          // 函数里 T 当正常类型用
    return a + b;
}

// 编译器自动推导（隐式实例化）
add(1, 2);        // T = int
add(1.5, 2.5);    // T = double

// 也可以显式指定 
add<int>(1, 2);   // 强制 T = int
```
