## 概述

状态位是用来告诉你[[stream]]现在能不能用的东西。

| 状态       | 含义    | 什么时候亮             |
| -------- | ----- | ----------------- |
| `good()` | 🟢 绿灯 | 一切正常              |
| `eof()`  | 🔵 蓝灯 | 读到结尾（End Of File） |
| `fail()` | 🟡 黄灯 | 操作失败（类型错、没数据）     |
| `bad()`  | 🔴 红灯 | 严重错误（系统崩了）        |

只要不是good，stream就不能使用了

## 例子

```
istringstream iss("100 abc");
int a, b;

iss >> a;             // ✅ 成功，a=100
cout << iss.good();   // 1（true，绿灯）

iss >> b;             // ❌ 失败！"abc"不是数字
cout << iss.fail();   // 1（true，黄灯亮了）
cout << iss.good();   // 0（false，绿灯灭了）
```
