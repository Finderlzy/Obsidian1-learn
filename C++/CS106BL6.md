## [[Associative  Container|map]]的补充

Map可以使用方括号。
如果方括号里面的key不存在，则直接新建一个key，value为0.

### 遍历Map的一种方法

```
Map<string, int> oneMap;
int sum = 0;

...... // -> 一些给oneMap的操作

for (string s : oneMap) // 这个s是一个新的变量
{
    sum += oneMap[s];
}
```

大致就是这样。
