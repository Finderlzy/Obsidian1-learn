## 概述

操作符用来让[[Stream|流]]的输入变得有趣

## 常见操作符

| 操作符                   | 作用      | 例子                                   |
| --------------------- | ------- | ------------------------------------ |
| `endl`                | 换行 + 刷新 | `cout << endl`                       |
| `flush`               | 只刷新     | `cout << flush`                      |
| `setw(n)`             | 设置宽度    | `cout << setw(10) << 123`            |
| `setprecision(n)`     | 小数位数    | `cout << setprecision(2) << 3.14159` |
| `fixed`               | 固定小数    | `cout << fixed << 3.14`              |
| `hex` / `dec` / `oct` | 进制切换    | `cout << hex << 255` → `ff`          |
| `left` / `right`      | 对齐方式    | `cout << left << setw(10) << "hi"`   |

## 展示

```
#include <iomanip>  // 操作符头文件

cout << setw(10) << 123 << endl;       // "       123"（右对齐补空格）
cout << setfill('0') << setw(5) << 7;  // "00007"
cout << hex << 255;                    // "ff"
```

## 关于endl换行

由于endl每次换行之后都会flush一次，这就使得在多次循环场景，endl效率很低。因此，在需要高性能的时候，要少用endl换行。如果想用，就用“\n”。
