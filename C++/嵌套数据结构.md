## 定义

顾名思义，嵌套数据结构就是一个容器的类型是另一个容器。
```
Map<string, Vector<int>> testMap;
```
这就是一个嵌套数据结构。

## 理解嵌套

可以这样理解这个Map：
      **testMap**

| key  | value（这样理解） |
| ---- | ----------- |
| li   | Vector li   |
| zhao | Vector zhao |
```
/* 可以这样理解
*/

testMap[li].add(666); 
li.add(666);
```

还有就是，**方括号相当于引用**。
上面第一行操作之后，vector的值会永久改变。
与之对应的是下面的操作。

```
Vector<int> temp = testMap[li];

这个操作只是把Vector li的值复制到temp里面，并不是让temp等于li的引用。
其实也很好理解，temp是Vector，不是Vector&。
```
