## [[Console Programs]]（控制台程序）

就是那个小黑框框，是用户与程序交流的平台。控制台通过打印文本在屏幕上，向用户输出信息。

控制台该如何获取用户输入的信息呢？getLine函数就是用来干这个的 **（返回string类型）**

那如果用户输入的是整型数据呢？这时候就需要转换数据类型了。

有时，需要不停地向用户发送信息，就可以用一般用不到的无限循环——==while（true）==。但是要记得添加条件break。

## [[Sequence Container|vector]]（向量或容器）
### 概述

可以理解为数组，但是有一个地方很不一样——数组在定义和使用的时候必须确定大小，但vector不用。当你往vector里面新增元素时，它会自动扩大空间，相当于**动态数组**。

### 操作
#### 头文件

想使用vector得先包含头文件
`#include <vector>`

#### 创建

`Vector<dataType> vectorName;`
`eg. Vector<int> vec;`

==Vector的第一个字母要大写==

#### 初始化

像数组一样，用方括号。

#### 增加元素

```
Vector<int> vec;
vec.add(12);
```

这时，vector会新增0号元素，元素类型是int型，值是12.

#### 删除元素

```
Vector<int> vec;
vec.add(12);
vec.remove(0);
```

==删除的时候，参数是索引，而不是元素值。这和增加的时候不一样==

## [[Grid]]（网格）
### 概述

这是斯坦福自创的数据结构，类似于二维数组。

通过行和列来索引

### 创建

和vector一样的创建方法方式。
```
Grid<dataType> gridName;
eg. Grid<int> world;
```
